Testing CSP Configuration

Open the browser console to check for CSP violations.

Steps to verify:

  1. Open http://localhost:3000 in your browser
  2. Open Developer Tools (F12)
  3. Go to the Console tab
  4. Look for any CSP violation errors (they appear in red)
  5. Check the Network tab for blocked Google Analytics requests
  6. Verify gtag/analytics scripts are loading successfully

Expected Result:

  • No CSP violations in console
  • Google Analytics scripts load from googletagmanager.com
  • Analytics requests to google-analytics.com succeed

Updated CSP Policy:

    script-src: 'self' 'unsafe-eval' 'unsafe-inline' https://www.googletagmanager.com https://www.google-analytics.com
    connect-src: 'self' https://www.google-analytics.com https://analytics.google.com https://www.googletagmanager.com
    img-src: 'self' data: https: https://www.google-analytics.com https://www.googletagmanager.com
