# SkidSpace Brand Assets

This folder contains the official SkidSpace brand assets based on the brand guide.

## Files

- `logo-icon.svg` - Primary icon logo (256x256) for app icons, favicons, map pins
- `logo-primary.svg` - Horizontal logo (640x160) for website headers, hero sections
- `map-pin.svg` - Standard map pin (blue)
- `map-pin-active.svg` - Active map pin (orange)

## Brand Colors

- **Blue**: #0B5FFF (Primary brand color)
- **Orange**: #FF8A1F (Accent color)
- **Ink**: #0B1220 (Text color)
- **Slate**: #475569 (Muted text)
- **Border**: #E2E8F0 (Borders)
- **Background**: #FFFFFF (White)
- **Muted Background**: #F8FAFC (Light gray)

## Typography

- **Font Family**: Inter, system-ui, sans-serif
- **Headings**: 600 weight
- **Body**: 400 weight
- **Buttons**: 500 weight

## Usage Rules

- No gradients
- No shadows on logos
- No recoloring icons
- No text inside pin icons
- Maintain minimum clear space around logos
