# SkidSpace Brand Assets

This folder contains the official SkidSpace brand assets based on the brand guide.

## Files

- `logo-icon.svg` - Expressive icon with dual-layer pin design (256x256) for app icons, favicons
- `logo-primary.svg` - Enhanced horizontal logo with dual-layer pin (640x160) for headers
- `system-icon.svg` - Clean system icon with stroke-based design for UI elements
- `map-pin.svg` - Standard map pin with stroke warehouse (blue)
- `map-pin-active.svg` - Active map pin with stroke warehouse (orange)

## Brand Colors

- **Blue**: #0B5FFF (Primary brand color)
- **Orange**: #FF8A1F (Accent color)
- **Ink**: #0B1220 (Text color)
- **Slate**: #475569 (Muted text)
- **Border**: #E2E8F0 (Borders)
- **Background**: #FFFFFF (White)
- **Muted Background**: #F8FAFC (Light gray)

## Typography

- **Font Family**: Inter, system-ui, sans-serif
- **Headings**: 600 weight
- **Body**: 400 weight
- **Buttons**: 500 weight

## Usage Rules

- No gradients
- No shadows on logos
- No recoloring icons
- No text inside pin icons
- Maintain minimum clear space around logos
